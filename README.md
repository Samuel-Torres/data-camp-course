# data-camp-course

Course notes and exercises.

## OpenAI Python example

1. Create a virtualenv (optional) and install deps:

   ```bash
   pip install -r requirements.txt
   ```

2. Set your API key (do not commit it):

   ```bash
   export OPENAI_API_KEY="your-key-here"
   ```

3. Run:

   ```bash
   python openai_request.py
   ```

If an API key was ever pasted into this repo, rotate it in the [OpenAI API keys](https://platform.openai.com/api-keys) dashboard.
